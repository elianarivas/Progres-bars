Barra de progreso

Mayormente utilizada para páginas donde se encuntrean distintos relatos, cuentos, poemas, etc.

Hoy día, Java es la plataforma que se ha posicionado como número uno de los Lenguajes de Programación y Desarrollo; esto, porque permite reducir costos, acortar los plazos de desarrollo, impulsar la innovación y mejorar los servicios de las aplicaciones. Aspectos determinantes en su gran aceptación. Una de las características de esta plataforma es AWT (ABSTRACT WINDOWING TOOLKIT), la cual proporciona múltiples clases para crear vistas GUI. AWT es casi similar a SWING, solo que la pantalla AWT es más simple que SWING. Por tanto, AWT, que en español significa “Kit de Herramientas de Ventana Abstracta”, es un conjunto de herramientas GUI (Interfaz Gráfica con el Usuario), diseñadas para trabajar con múltiples plataformas. Interfaz de usuario, y sistema de ventanas independiente de la plataforma original de Lenguaje de Programación Java. Con relación a Swing, biblioteca gráfica para Java; esta incluye widgets para interfaz gráfica de usuario tales como cajas de texto, botones, desplegables y tablas. El paquete Swing es el nuevo paquete gráfico que ha aparecido en la versión 1.2 de Java, compuesto por un amplio conjunto de componentes de interfaces de usuario que funcionen en el mayor número posible de plataformas. De allí que, AWT y Swing son dos de las bibliotecas disponibles en el paquete estándar de Java para la creación de interfaces gráficas de usuario. Mientras que AWT descansa sobre el SO para la creación de los componentes gráficos, Swing abstrae el SO y proporciona su propia colección de componentes gráficos.
